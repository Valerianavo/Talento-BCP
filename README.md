# 🚀 Talento BCP

> Plataforma web inteligente de gestión y descubrimiento de talento interno del BCP
> Desarrollado para la Hackathon 2026 +ChicasTec por el equipo **4byte**

---

## 🎯 Propósito del proyecto

**Talento BCP** es una plataforma diseñada para conectar practicantes con líderes del banco, permitiendo visualizar, filtrar y evaluar talento interno de forma rápida, estructurada y en tiempo real.

El objetivo es optimizar la búsqueda de perfiles adecuados dentro del BCP, al mismo tiempo que se brinda a los practicantes un espacio para mostrar su crecimiento profesional.

---

## 🧠 Problema que resolvemos

En organizaciones grandes como el BCP, identificar talento interno relevante puede ser lento y poco eficiente.

**Talento BCP resuelve esto mediante:**

* Centralización de perfiles de practicantes
* Búsqueda inteligente por filtros avanzados
* Evaluación estructurada por líderes
* Acceso inmediato a información relevante del talento

---

## ⚙️ Tecnologías utilizadas

* ⚛️ React (Create React App)
* 🔥 Firebase (Authentication + Firestore en tiempo real)
* 🎨 Bootstrap + CSS personalizado
* 🌐 GitHub Pages (deploy)

---

## ✨ Funcionalidades principales

### 👥 Gestión de usuarios

* Registro e inicio de sesión
* Roles diferenciados:

  * 🧑‍🎓 Practicante (sube y gestiona su perfil)
  * 🧑‍💼 Líder (busca, evalúa y analiza talento)

---

### 📂 Catálogo de practicantes

* Tarjetas dinámicas de perfiles
* Vista previa de información
* Acceso a perfil completo
* Actualización en tiempo real con Firebase

---

### 🔎 Búsqueda avanzada

* Filtros por:

  * Skills / tags
  * Áreas de experiencia
  * Ubicación
  * Idiomas
  * Tiempo de experiencia
  * Formación académica
  * Proyectos
  * Talentos favoritos

---

### 📊 Dashboard del líder

* Visualización centralizada del talento
* Métricas de evaluación de practicantes
* Herramientas de análisis interno
* Control total del ecosistema de talento

---

### 🤖 Funcionalidades inteligentes

* Chatbot de ayuda para líderes
* Búsqueda por voz para agilizar filtrado de perfiles
